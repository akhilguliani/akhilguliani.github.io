Study And Implementation Of Real Time Operating Systems For 8-bit Microcontrollers

Report on Internship at IIT Delhi in summer 2011. I studied Real Time operating systems (RTOS) and surveyed the possible RTOS options available for a TinyAVR microcontrollers. Chose to use Femto OS. Built the reference board and implemented an I2C driver for Femto OS for the design.
