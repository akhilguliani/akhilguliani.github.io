Device File Virtualization

Developed as part of a team a proof of concept for device file virtualization for Palacios Virtual Machine Monitor. The concept allows a Linux guest to access the device files of the Linux host. Our implenattion was pulled into the main repository available publically at this link. This project started as part of a course and was developed further during an independent study.
